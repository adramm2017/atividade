Usar Versionamento e Git para entrega Usar o Xampp para teste Organizar o projeto com diretórios conforme exemplos Fazer um commit de cada alteração em uma ferramenta Git. Github ou gitlab ou bitbucket e openshift Importante: Criar um repositorio com um README.md, faça um clone para o seu computador, que está disponível na aula 3, incluir arquivos na pasta que foi clonada. A cada alteração faça um add e um commit com a mensagem de forma correta para facilitar o changelog. Faça um push após algum tempo para enviar as alterações para o servidor Git. Ao final, crie uma TAG e envie para o servidor central. Entrega: através de um link do repositório onde o gerador será mantido.
